What is the Document Object Model (DOM)?

DOM = D = Document. O = Object. M = Model. DOM is the data represent objects and the structure on web pages. Its represent HTML and XML Document as like tree structure.Documents are arranged in a tree. Document has some methods which the document structure, style, and content can be changed using the programming language JavaScript.

How do you select an element from the DOM?

There are many ways to select elements of DOM like id Selector, class Selector, and tag name Selector etc. example: I have an id, or class, = "myName" id selector = document . getElementById( ' myName ') class selector = document . getElementByClassName ("myName")

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to using event propagation (bubbling) to handle events.It's very useful to handle the event parent and child relations.We should always set the parent elements for event delegation. Because it saves memory and dynamically adds elements and sets handler.

How do you manipulate an element's attributes and styles using the DOM?

I have almost two ways to change elements attributed and style like
classList and setAttribute() method.
classList object change only elements class.
other attributes like id, href, src etc ... use to setAttribute() method.
setAttribute() method can receive two arguments first attribute name second value.
